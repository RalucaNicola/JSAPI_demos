Earthquakes between 2008 and 2018

Earthquake data containing more than 245000 earthquakes with a magnitude higher than 2.5, which occurred between January 2008 and start of May 2018. The depth of the earthquake is inverted and displayed as altitude. Earthquakes are displayed as circles with magnitude between 2.5 and 4.3, between 4.31 and 7.0 and higher than 7.01. The size is proportional to the magnitude.

Original data from USGS downloaded on May 8, 2018. This prototype was built with ArcGIS API for JavaScript. The code for this app is on Github.
